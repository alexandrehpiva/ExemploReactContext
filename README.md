## Context + Hooks no React

Para resolver o problema de "Prop Drilling" no React (passar um state de componente a componente através das props até chegar no componente desejado).

Com o context, podemos passar dados para a nossa árvore de componentes sem sofrer com "Prop Drilling".

### Context + Hooks in React

To solve or problem of "Prop Drilling" in React (pass a state from component to component through the properties until reaching the selected component).

With context we can pass data down on our component tree without Prop Drilling.
